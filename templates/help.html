Help

Input Batches - The number of days to be predicted on
Output Batches - The number of predicted days
Training data - The data that the LSTM trains off
Prediction data- The data that is used to generate predictions
Epochs- Number of times the LSTM runs throught the training data
Activation Function Layer - Activation function decides, whether a neuron should be activated
Loss Function - The difference between the actual output and the predicted output from the model.
Stacked Layers - Allows LSTM layer outputs to be used as an input to a subsequent LSTM layer.
